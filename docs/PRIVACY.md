# Privacy

OpenRouterDesk is local-first.

## Default behavior

- No telemetry.
- No cloud account.
- No upload of credentials, configuration, inventory, or topology.
- Update checks and release downloads are explicit and documented.

## Exports

Reports default to masked MAC addresses, sanitized hostnames, and excluded credentials. The user must explicitly choose to include sensitive fields.

## Future telemetry

Any future telemetry proposal requires an ADR, opt-in consent, a public schema, data minimization, retention rules, and a way to inspect and delete collected data. It is not part of the MVP.
